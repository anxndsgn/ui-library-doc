## Props

### SelectTrigger

| Prop        | Type                  | Default     | Description                                  |
| ----------- | --------------------- | ----------- | -------------------------------------------- |
| `size`      | `"sm" \| "default"`   | `"default"` | Size of the trigger.                         |
| `className` | `string`              | -           | Adds custom classes to the trigger.          |

### SelectContent

| Prop                   | Type                                                   | Default    | Description                                        |
| ---------------------- | ------------------------------------------------------ | ---------- | -------------------------------------------------- |
| `side`                 | `"bottom" \| "top" \| "left" \| "right"`               | `"bottom"` | Preferred side of the trigger to display on.       |
| `sideOffset`           | `number`                                               | `4`        | Pixel offset from the trigger edge.                |
| `align`                | `"start" \| "center" \| "end"`                         | `"center"` | Horizontal alignment relative to the trigger.      |
| `alignOffset`          | `number`                                               | `0`        | Pixel offset along the alignment axis.             |
| `alignItemWithTrigger` | `boolean`                                              | `true`     | Aligns the popup width with the trigger width.     |
| `className`            | `string`                                               | -          | Adds custom classes to the popup.                  |
