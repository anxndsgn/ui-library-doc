## Props

### ScrollArea

| Prop        | Type        | Default | Description                                  |
| ----------- | ----------- | ------- | -------------------------------------------- |
| `className` | `string`    | -       | Adds custom classes to the root container.   |

### ScrollBar

| Prop          | Type                       | Default      | Description                                  |
| ------------- | -------------------------- | ------------ | -------------------------------------------- |
| `orientation` | `"vertical" \| "horizontal"` | `"vertical"` | Orientation of the scrollbar.                |
| `className`   | `string`                   | -            | Adds custom classes to the scrollbar.        |
