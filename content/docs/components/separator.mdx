## Props

| Prop          | Type                           | Default        | Description                                  |
| ------------- | ------------------------------ | -------------- | -------------------------------------------- |
| `orientation` | `"horizontal" \| "vertical"`   | `"horizontal"` | Direction of the separator.                  |
| `className`   | `string`                       | -              | Adds custom classes to the separator.        |
